# TwinDimensions_Sokoban | 双界推箱子

本项目是一款基于 **Unity** 开发的创新解谜游戏。核心机制围绕“双生维度”展开，通过严格的量子纠缠逻辑，挑战玩家的空间逻辑思维。

## 🧩 核心玩法 (Gameplay Mechanics)

游戏的核心在于 **“绝对同步”**。玩家控制的两个小球分别处于光、影两个维度：
- **同步移动**: 按下移动键时，两个维度的角色会同时尝试向目标格移动。
- **严格碰撞约束**: 这是一个“牵一发而动全身”的机制。若其中一个维度的路径上有障碍物（墙壁、箱子边缘等），**两个维度的移动动作都会被判定失效**。玩家必须利用这一特性，通过在一个维度“撞墙”来调整另一个维度的相对位置。
- **胜利条件**: 只有当两个维度的 **所有目标点 (Target Pads)** 同时被箱子覆盖时，关卡才会判定胜利并解锁 **Next Level**。

## 🛠️ 自定义编辑器工具 (Level Editor Toolset)

为了提高开发效率，本项目内置了一套“所见即所得”的编辑器扩展工具，支持在 Unity Scene 视图中直接进行关卡创作。

### 1. QuickLevelBrush (快捷笔刷)
通过监听编辑器事件，实现了无需拖拽 Prefab 即可快速布置关卡的功能：

| 快捷组合键 | 放置物体 | 预制体类型 |
| :--- | :--- | :--- |
| **Shift + 左键** | 光界障碍 | Wall_LIGHT |
| **Shift + 右键** | 影界障碍 | Wall_SHADOW |
| **B + 左键** | 量子箱子 | Box |
| **T + 左键** | 目标感应垫 | Target Pad |

### 2. Level Exporter (关卡导出系统)
- **智能扫描**: 脚本支持全场景扫描，根据物体坐标自动判定所属维度（Light/Shadow）。
- **JSON 持久化**: 在 Inspector 面板中输入关卡名（如 `Level_1`），通过右键菜单执行 `Export`，系统会将所有物体坐标、标签及维度属性序列化为 JSON 文件。
- **解耦加载**: 游戏运行或重置（R 键）时，`LevelLoader` 会实时读取对应的 JSON 配置文件，实现物理世界的零延迟重建。

## 🌟 技术亮点 (Technical Highlights)
- **数据驱动架构**: 关卡内容与场景完全解耦，支持不限量关卡扩展而无需新建 Scene。
- **空间过滤算法**: 在导出时通过 X 轴阈值（xThreshold）自动进行空间分拣，解决了多维物体坐标冲突问题。
- **鲁棒性清理**: 针对关卡重置设计了强制清理与自检逻辑，防止因层级残留导致的内存泄漏或渲染重叠。

## 🎮 玩家操作
- **WASD / 方向键**: 操控双维度角色。
- **R 键**: 瞬间重置当前关卡布局。
- **UI 切换**: 主菜单支持 Level 1 - Level 3 的动态选关。
